How to use the module: Sale Target

Overview

The Odoo behavior has the limitation of defining sale target on each salesperson unless we go one by one and sum the sale confirmed ad invoice confirmed for example, so it takes so much time to have each target on the salesperson.
This is the reason why we build this module to support this need. Meaning that you can setup the salesperson target from start date to end date, and you will have the summary of the target reach accordingly.

Features
Ability to setup sale target in each salesperson
Ability to check the reporting salesperson on each target
How it works
It will be availale on youtube soon

To make this module work in your database. You have to follow these steps as below:


Step 1: Purchase this module

Step 2: Copy and paste this module to your folder custom addons

Step 3: Rebuild your server

Step 4: Install app

After install the app, then you will be able to use the feature in this module as the below demonstration:


Step 1: Assign the permission to user which you want to allow to configure sale target


Step 2: After the user has that permission, he/she will be able to see another menu in sale configuration called Sale Target. For that menu, manager can setup the new sale target as below screenshot


Step 3: Click create to setup the new sale target, then you will see as below screenshot: There are two condition of the sale target which is “Sale Order Confirmed, Invoice Confirmed”. Meaning that it will count only if the SO confirmed and invoice confirmed. Target amount is to target the value of your sale team, reach amount is to tell what amount your salesperson achieve and different amount is the different between target and reach amount.


Step 4: At the end, you will have the reporting of sale target to see the summary of each sale person as below screenshot
